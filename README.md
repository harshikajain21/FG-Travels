FG Travels 🚗

FG Travels is a car rental web application designed to help users browse available vehicles and connect directly for bookings. The platform provides a simple and user-friendly interface for exploring rental cars and contacting the business through WhatsApp.

🌟 Features
🏠 Home Page
Attractive landing page introducing FG Travels.
Easy navigation to different sections of the website.
🚘 Fleet Section
Displays all available rental cars.
Shows detailed information about each vehicle.
Currently includes 5 rental car options.
ℹ️ About Section
Information about FG Travels and its services.
Helps customers understand the business and offerings.
💬 WhatsApp Integration
Direct WhatsApp contact button.
Allows customers to quickly inquire about bookings and rental details.

⚙️ Installation
Clone the repository:
git clone https://github.com/your-username/FG-Travels.git
Navigate to the project folder:
cd FG-Travels
Install dependencies:
npm install
Start the development server:
npm run dev
